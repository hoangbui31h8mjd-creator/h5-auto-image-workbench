# 原型编辑器

本目录用于沉淀未来工作台中“原型阶段可编辑”的设计方案和参考文件。

## 产品判断

新工作台的关键不是直接生成最终长图，而是先生成一个可编辑 H5 原型。

建议流程：

```text
需求输入
-> 生成原型
-> 人修改原型
-> 确认后生图
-> 输出长图
```

## 第一阶段编辑能力

优先支持组件化编辑，不做完整自由画布。

需要支持：

- 修改文案
- 替换组件
- 调整模块顺序
- 修改颜色主题
- 替换图片或图片占位
- 局部重生成某个模块
