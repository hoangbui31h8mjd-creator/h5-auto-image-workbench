# H5 Agent 切图 Skill 执行版 v0.1

> 适用阶段：Codex 本地 H5 生成 Agent  
> 目标：根据 HTML 原型的模块结构，规划 1-N 张原型切片，并为后续 Image2 分段生图预留参考带空间。

---

## 1. 使用边界

本 Skill 只负责“原型图怎么切”，不负责组件长什么样，也不负责最终生图风格。

它应该用于：

- 根据完整 HTML 原型和渲染后的长图高度，决定切成几段。
- 根据模块边界、自然留白、低信息区选择切点。
- 避免切断主体内容。
- 为第 2 段及之后的顶部参考带预留高度预算。

它不应该用于：

- 固定规定必须切成 3 段。
- 在画面上画安全区、灰条、切口线或文字说明。
- 决定 Image2 的视觉风格、材质、光影和文案语言。

---

## 2. 当前核心规则

```text
切片数量：不固定，按页面内容自动得到 1-N 段
画布宽度：读取组件注册表或实际渲染图宽度，默认 828px
模型比例上限：1:3
参考带高度：80px
单张原型切片高度预算 = 当前原型图宽度 × 3 - 80
```

解释：

- Image2 单次输入不能超过 1:3。
- 第 2 段及之后，会在当前原型切片顶部额外拼上上一段生成图底部的 80px 参考带。
- 所以切片阶段不能把原型切片本身切到 `宽度 × 3`，必须提前减掉 80px。

---

## 3. 切点选择逻辑

从上往下循环切：

1. 以当前段起点为 `segment_start`。
2. 计算本段最大允许底线：`segment_start + 当前宽度 × 3 - 80`。
3. 如果这个位置落在模块中间，不硬切。
4. 回退到这条底线之前最近的可靠切点。
5. 从新切点继续往下重复，直到完整页面切完。

可靠切点优先级：

```text
1. data-cut-after="preferred"
2. 模块结束位置
3. 模块之间的自然留白
4. 背景简单、信息密度低的位置
5. 最后才在最大高度附近兜底
```

---

## 4. 禁切区域

以下内容默认不能被切断：

- 标题文字。
- CTA 按钮。
- 任务卡片。
- 奖励卡片。
- 视频框。
- 表格行。
- 排行榜单行。
- FAQ 单条。
- 规则段落中间。
- 人物、商品、奖品等核心视觉主体。

如果模块本身特别高，已经超过单段高度预算，才允许在模块内部找更细的低信息子边界，例如卡片组之间、列表项之间、段落组之间。

---

## 5. 与组件 Skill 的关系

组件 Skill 负责在 HTML 中提供机器可读标记：

```html
<section data-section="task_list" data-component="task_list" data-no-cut="true">
  ...
</section>

<div class="module-gap" data-module-break="task_list" data-cut-after="preferred"></div>
```

切图 Skill 负责读取这些标记并做切点决策。

简单说：

```text
组件 Skill：告诉机器哪里是模块、哪里不要切、哪里适合切。
切图 Skill：根据这些信息真正决定切在哪里。
```

---

## 6. 输出契约

切图规划应输出机器可读信息：

```json
{
  "canvas_width": 828,
  "model_max_ratio": "1:3",
  "model_max_height": 2484,
  "reference_band_height": 80,
  "raw_slice_height_limit": 2404,
  "cut_positions": [2400, 4688],
  "slices": [
    {
      "index": 1,
      "y0": 0,
      "y1": 2400,
      "height": 2400,
      "reason": "module_boundary"
    }
  ]
}
```

注意：上面的数字只是示例。实际值必须按当前原型图宽度和页面高度计算。

---

## 7. 质检规则

切片后必须检查：

- 是否没有固定切片数量。
- 是否每张原型切片高度不超过 `当前宽度 × 3 - 80`。
- 是否第 2 段及之后加上 80px 参考带后仍不超过 1:3。
- 是否没有切断 `data-no-cut="true"` 模块。
- 是否优先使用 `data-cut-after="preferred"` 或模块边界。
- 是否没有在最终可见画面中出现“安全区、80px、切口、接缝、续画、拼接”等生产词。

---

## 8. 一句话原则

```text
宁可多切一段，也不要切断主体内容；宁可提前回退到上一模块末尾，也不要卡在 1:3 极限处硬切。
```
