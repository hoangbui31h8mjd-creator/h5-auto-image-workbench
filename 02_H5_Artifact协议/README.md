# H5 Artifact 协议

本目录用于定义新工作台的核心产物结构。

未来一个 H5 不应该只是一张最终长图，而应该是一个完整的 H5 Artifact。

## 一个 H5 Artifact 至少包含

```text
用户需求
补问记录
原型 JSON
组件列表
文案内容
图片素材
设计规范引用
生图 prompt
生图参数
切片结果
最终长图
版本记录
```

## 当前样例

`样例_已跑通/` 中放的是旧链路已跑通产物中的关键结构文件：

```text
01_blueprint.json
02_prototype.html
slices_manifest.json
run_summary.json
```

这些文件不是最终协议，只是下一步抽象协议的事实参考。
