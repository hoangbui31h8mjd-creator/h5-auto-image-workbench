## 模块边界

| 模块 | 负责 | 不负责 |
|---|---|---|
| Codex 产品包 | 用户入口、输入检查、执行说明、Skill 和知识库组织 | 保存密钥、默认执行历史案例 |
| Codex 研发工作区 | 调试、规则迭代、失败案例、历史问题沉淀 | 作为用户默认入口 |
| 本地 Python Pipeline | 原型生成、渲染、切片、生图调用、拼接、质检 | 替用户补全不存在的业务需求 |
| 旧 Dify/ComfyUI 链路 | 历史验证、回看、排查旧问题 | 当前产品入口 |

## 验收级别

| 级别 | 判定条件 | 当前状态 |
|---|---|---|
| L0 目录可用 | 用户能找到产品包入口，旧链路不混在当前区 | 已完成 |
| L1 输入门禁 | 缺 KV 或内容信息时会先问用户；特殊要求不作为必填项 | 规则已写入，待新窗口实测 |
| L2 原型与切图 | 本地 runner 可产出 HTML、原型图和切片 | 已有代码能力，需用当前产品包任务复测 |
| L3 mock 全链路 | mock 生图、拼接、质检跑通 | 已有能力，需用新目录复测 |
| L4 真实模型链路 | 真实 Image2 成功生成最终长图 | 待持续验证 |
| L5 产品化 | 独立打包、一键运行、多人可用、密钥安全 | 未达到 |

## 安全边界

- 不保存 API key。
- 不保存临时 tunnel 地址为默认值。
- 不把历史案例当当前任务。
- 不把旧 Dify/ComfyUI 链路描述成当前产品主入口。
