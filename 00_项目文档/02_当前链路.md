# 当前链路

本文描述当前阶段的 Codex 本地 H5 Agent 链路。

## 用户侧链路

```text
用户打开 01_H5_Agent产品包
-> 用户说“帮我生成一张 H5”
-> Codex 读取 00_AGENT入口说明.md
-> Codex 检查 01_用户输入区/当前任务
-> 缺 KV / 内容信息则补问
-> 输入齐全后执行本地 H5 Pipeline
-> 输出原型、切片、最终图和质检报告
```

## 能力链路

```text
KV / 头图
+ 内容信息
+ Skill 默认规则
+ 用户特殊要求（可选）
+ Skill
+ 知识库
-> 需求理解和页面蓝图
-> HTML 原型
-> 原型自查与修复
-> 渲染完整原型图
-> 按模块边界切片
-> 生成分段生图提示词
-> 调用 Image2 或 mock 生图
-> 80px 参考带续接
-> 重叠羽化拼接
-> 输出最终 H5 长图和质检报告
```
