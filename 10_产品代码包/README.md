# H5 Agent 工作台代码包说明（公开版）

这个文件夹在公开仓库里只保留代码包说明，不再放完整工程源码。

我本地已经做过一版半成品工程，里面包括 Web 工作台、本地 daemon、H5 Artifact 协议包、Agent 事件契约、旧 runner 适配层和若干 mock 接口。但这部分现在还属于个人本地原型，很多细节没有整理成适合公开维护的版本，所以先不作为作业材料全量开源。

## 公开版保留什么

公开仓库主要保留这些内容：

- 产品定义和作业思路，让第一次打开的人能知道我要解决什么问题。
- H5 Artifact 协议、组件库、设计规范和本地 Pipeline 说明，用来展示工作流是怎么拆的。
- 已跑通样例、验收记录、视觉预览图和界面截图，用来证明这个方向不是停留在想法层。
- 这份代码包说明，用来交代完整工程在本地大概包含哪些模块。

## 完整工程暂不公开什么

暂不公开的部分包括：

- Web 工作台源码。
- 本地 daemon 和接口实现。
- Agent 事件协议包的具体实现。
- 旧项目 runner 适配层。
- 本地调试脚本、缓存、依赖和环境配置。

## 完整工程的大致结构

本地版本按下面这个方向组织：

```text
apps/web/                  工作台前端界面
apps/daemon/               本地后端、Agent 调度、runner 调用
packages/h5-artifact/      H5 Artifact 核心协议
packages/agent-contract/   Agent 输入输出契约
skills/                    H5 生成、编辑、生图相关 Skills
design-systems/            内部 H5 设计规范
examples/                  示例任务和示例 Artifact
```

## 这样处理的原因

这次作业最重要的是证明我能把一个真实需求拆开，并且设计出可以落地的 AI 工作流。完整源码如果还没整理干净就直接公开，反而会让读者把注意力放到未完成的工程细节上。

所以公开版先展示需求、流程、结构、协议、样例和界面结果；完整源码等后续稳定、脱敏、补齐说明以后，再考虑整理成真正适合公开维护的版本。
