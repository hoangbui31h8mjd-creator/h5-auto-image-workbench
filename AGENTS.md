## 当前项目主线

本项目目标是构建：

```text
组件化 H5 Agent 生产工作台
```

核心闭环：

```text
需求输入
-> 生成可编辑 H5 原型
-> 人修改原型
-> 确认后生图
-> 导出长图和项目包
```

## 重要边界

- 不要把旧项目整体搬进来。
- 不要把 `05_Agent流程/` 当作新项目默认入口。
- 不要复活旧 Dify / ComfyUI 链路作为当前产品主入口。
- 不要把旧 runner 输出继续写回旧项目目录作为长期方案。
- 第一阶段不做多人协作、账号体系、商业化平台和完整自由画布。

旧项目中的 Agent 规则、组件库、设计规范和 runner 链路可以参考或抽取。
