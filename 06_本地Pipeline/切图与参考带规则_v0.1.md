# 切图与参考带规则 v0.1

这份规则用于本地 H5 Agent 接 Image2 分段生图前的切片规划。

## 一句话

切图不是固定切几份，而是按 H5 模块结构切；每一刀尽量落在完整模块结束处，同时给后续 `80px` 参考带预留模型比例空间。

## 当前参数

```text
模型最大比例：1:3
默认画布：828px，来自积木/Figma 规范
参考带高度：80px
```

计算方式：

```text
model_max_height = 当前原型图宽度 × 3
raw_slice_height_limit = model_max_height - 80
```

原因：第 2 段及之后会在当前原型切片顶部追加上一段底部 `80px` 参考带。如果原型切片本身已经接近 1:3，再加参考带就会超比例，所以切图阶段要提前扣掉这 80px。

## 切点怎么选

优先级：

```text
1. HTML 里的 module-gap / data-module-break
2. 模块结束位置
3. 子组件结束位置，比如任务行、榜单行、奖励卡组结束
4. 自然留白或低信息区域
5. 最后才在最大高度附近兜底
```

禁止切断：

- 标题和正文组合。
- 按钮。
- 任务卡。
- 奖励卡。
- 视频框。
- 表格行。
- 排行榜行。
- FAQ 单条。
- 最终 CTA。

## 和生图接龙的关系

第 1 段输入：

```text
KV + 原型切片 1
```

第 2 段及之后输入：

```text
KV + 上一段生成图底部 80px 参考带 + 当前原型切片
```

参考带只是机器上下文，不是页面模块；HTML 原型里不能显示“安全区、80px、切口、接缝”等生产文字。

## 和最终拼接的关系

最终拼接时，后一段顶部的参考带应与上一段底部同一块区域重叠，并做羽化过渡。

如果模型改动了这 80px 参考带，应在拼接前优先用上一段真实底部参考带锁回去，再羽化。
