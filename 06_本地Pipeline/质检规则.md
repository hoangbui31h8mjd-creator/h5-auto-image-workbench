# 质检规则

本目录用于沉淀 H5 Agent 的质量检查规则。

## P0 质检

### HTML 原型

- HTML 是否完整。
- 是否能被浏览器渲染。
- 是否包含主要业务模块。
- 是否把用户提供的头图/KV 放在 HTML 首屏顶部。
- 是否出现“安全区、40px、80px、切口、接缝、续画”等禁用可见文字。
- 是否在首屏以外错误重复嵌入 KV，或使用外链资源。

### 切片

- 是否支持 1-N 段。
- 单段高度是否符合模型比例限制：`当前原型图宽度 × 3 - 80`。
- 第 2 段及之后加上 `80px` 参考带后，整体是否仍不超过 1:3。
- 是否切断标题、按钮、任务卡、奖励卡、排行榜、FAQ。
- 切点是否落在模块间留白或低信息区。

### 生图

- 是否参考了原型切片布局。
- 是否参考了 KV 风格。
- 是否出现明显拉伸、压缩、错位。
- 是否出现错误语言或乱码文字。

### 拼接

- 是否有明显白线、灰条、断层。
- 是否出现重复标题或模块错位。
- 上下段风格是否连续。

## P1 质检

- OCR 检查关键文案。
- 接缝区域图像差异检测。
- 模块数量和蓝图一致性检查。
- 输出成本和耗时记录。
